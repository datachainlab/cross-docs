# Demo

This part explains how to set up Cross-chain environments and execute demos.

---

## Tendermint and Hyperledger Fabric

This demo shows the interoperability between Tendermint and Hyperledger Fabric in [fabric-tendermint-cross-demo](https://github.com/datachainlab/fabric-tendermint-cross-demo).

### Setup network
Read [Setup Network](https://github.com/datachainlab/fabric-tendermint-cross-demo/tree/main/demo#setup-network).

### Execute Cross-chain demo
The [sample scenario script](https://github.com/datachainlab/fabric-tendermint-cross-demo/blob/main/demo/scripts/scenario/sample-scenario) is ready. Read [About sample-scenario](https://github.com/datachainlab/fabric-tendermint-cross-demo/tree/main/demo#about-sample-scenario).

---

## Ethereum to Ethereum (Anvil)

This demo demonstrates an **ERC20 Atomic Swap** between two local Ethereum networks (Anvil) in [ethereum-cross-demo](https://github.com/datachainlab/ethereum-cross-demo).

### Execute Cross-chain demo
You can execute a cross-chain atomic swap of ERC20 tokens by running the [demo script](https://github.com/datachainlab/ethereum-cross-demo#testing-the-demo).
